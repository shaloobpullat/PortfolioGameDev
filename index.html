3D Car Parking Game

This 3D car parking simulator features progressively challenging levels and realistic driving mechanics. I designed 7 detailed levels with increasing difficulty and implemented accurate vehicle physics using Wheel Colliders. To enhance immersion, I introduced AI-controlled traffic and utilized Unity's Spline Tool to align streetlights and barriers realistically along the roads. The game was carefully optimized for smooth performance on mobile platforms, ensuring a polished and engaging user experience. Developed using Unity and C#.

2D Platformer Game

2D Platformer Game is a 2D tile-mapped platformer featuring custom sprites, physics-based puzzles, and responsive player controls. I designed engaging levels using Unity’s Tilemap system and added interactive elements like swinging platforms with Hinge Joint 2D. To ensure smooth and satisfying movement, I implemented precise jump and collision mechanics using Unity's 2D physics engine. The game also includes dynamic sound effects and player feedback elements that enhance the overall polish and playability. This project highlights my skills in 2D game development, level design, and gameplay feel using Unity and C#.

3D Soldier Rescue Game

Soldier Rescue is a stealth-action game where players infiltrate hostile zones to rescue captured soldiers, combining tactical gameplay with immersive storytelling. I designed the levels to encourage strategic movement and cover-based stealth, with multiple enemy AIs patrolling via NavMesh and custom agent logic. The game features cinematic cutscenes created using Unity's Timeline and Cinemachine to drive the narrative. I implemented responsive shooting mechanics, smooth combat animation blending through the Animator system, and immersive sound and visual effects to heighten the experience. This project demonstrates my ability to blend gameplay systems, AI behavior, and cinematic presentation using Unity and C#.
